Front-end Tech Stack
Includes HTML, CSS, and JavaScript responsible for the structure, style, and interactive features of web pages.
Back-end Tech Stack
Comprises server-side logic and data management, often using languages like Node.js, Python, or Java.
Database
Used for storing and managing data, with options such as MySQL, MongoDB, or PostgreSQL.
Version Control
Facilitates storing and managing code changes for team collaboration, commonly using systems like Git.
Web Services
Involves establishing communication with other applications and services through URLs and APIs.
Front-end Frameworks
Frameworks like React, Angular, or Vue.js aid in building interactive and visually appealing web applications.
Back-end Frameworks
Frameworks such as Express (Node.js), Django (Python), or Spring (Java) assist in building server-side logic.
